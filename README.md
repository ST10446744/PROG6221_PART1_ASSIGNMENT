# 🔐 Cybersecurity Awareness Chatbot

## 📌 Introduction

Cybersecurity threats have become increasingly common in South Africa, affecting individuals, businesses, and government institutions. Many users fall victim to cyberattacks such as phishing, malware, and social engineering due to a lack of awareness and understanding of online safety practices.

This project presents a **Cybersecurity Awareness Chatbot**, a console-based application developed in C#. The chatbot is designed to educate users through interactive conversation, helping them understand how to identify and avoid common cyber threats.

---

## 🎯 Purpose of the Application

The purpose of this chatbot is to:

* Educate users about cybersecurity risks
* Provide basic guidance on staying safe online
* Simulate real-life scenarios through conversation
* Promote awareness of safe digital practices among South African citizens

---

## ⚙️ Features

* 🔊 Voice greeting using a WAV file
* 🖥️ ASCII art interface for visual enhancement
* 💬 Interactive conversation with user input
* 🧠 Predefined responses for cybersecurity topics such as:

  * Password safety
  * Phishing scams
  * Malware and viruses
  * Social engineering
  * Safe browsing practices
* ⚠️ Input validation and error handling
* 🎨 Enhanced console UI with colours and formatting

---

## 🛠️ Technologies Used

* C# (.NET Console Application)
* System.Media (for audio playback)
* GitHub (version control)
* GitHub Actions (CI workflow)

---

## ▶️ How to Run the Application

1. Open the project in Visual Studio
2. Build the solution
3. Run the program
4. Enter your name and interact with the chatbot
